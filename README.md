# 個人房貸試算器_s1131547

## 功能介紹
* **房貸試算核心**：輸入房屋總價、自備款、貸款利率與年限，精準計算出「貸款總金額、每月應繳金額、首期利息/本金、總利息支出與總還款金額」。計算邏輯採用標準的「本息平均攤還法」。
* **輸入驗證與防呆機制**：有效攔截非法輸入（如英文字母、特殊符號或空白），並跳出警告視窗提示，確保程式不會因錯誤輸入而崩潰。
* **寬限期計算**：選填「寬限期」，計算前幾年僅繳納利息、期滿後剩餘本金平均攤還的真實財務負擔。

* **格外功能**:
  
  **1. 動態單位連動與防呆清空**：

  自備款提供「比例 (%)」與「金額 (元)」兩種輸入模式。切換選項時，旁邊的單位提示會自動變更，並同步清空輸入框，防止單位錯亂導致的計算錯誤。
  
  **2. 一鍵清除與預設範例**：

  點擊清除按鈕不僅會清空運算結果，還會自動載入一組合理常見的預設值（如 1500萬房價、20%自備款、30年期），並自動將游標移回「房屋總價」輸入框，大幅提升使用者操作的順暢度。

  **3. 數值格式化與防誤觸介面**：

  所有計算結果皆自動加上千分位逗號，並精確四捨五入至小數點後兩位 (NT$)，符合真實金融情境。同時將結果輸出區的文字框設定為 `ReadOnly`，防止使用者誤改計算結果；並於版面標註計算基準，讓整體介面更直觀專業。
